1.1 Interpolação e projeção

Seja dado um intervalo I=[x0,x1]⊂ℝ, x0≠x1. O espaço vetorial das funções lineares em I é definido por

P1⁢(I):={v:v⁢(x)=c0+c1⁢x,x∈I,c0,c1∈ℝ}. (1.1)

Observamos que dado v∈P1⁢(I), temos que v é unicamente determinada pelos valores α0=v⁢(x0) e α1=v⁢(x1). Como consequência, existe exatamente uma única função v∈P1⁢(I) para quaisquer dados valores α0 e α1. Desta observação, introduzimos a chamada base nodal {φ0,φ1} para P1⁢(I), definida por

φj⁢(xi)={1,i=j,0,i≠j, (1.2)

com i,j=0,1. Com esta base, toda função v∈P1⁢(I) pode ser escrita como uma combinação linear das funções φ0 e φ1 com coeficientes α0 e α1 (graus de liberdade), i.e.

v⁢(x)=α0⁢φ0⁢(x)+α1⁢φ1⁢(x1). (1.3)

Além disso, observamos que

φ0⁢(x)=x1-xx1-x0,φ1⁢(x)=x-x0x1-x0. (1.4)

Uma extensão do espaço P1⁢(I) é o espaço das funções lineares por partes. Dado I=[l0,l1], l0≠l1, consideremos uma partição (malha) de I com n+1 pontos ℐ={l0=x0,x1,…,xn=l1} e, portanto, com n subintervalos Ii=[xi-1,xi] de comprimento hi=xi-xi-1, i=1,2,⁢…,n. Na malha ℐ definimos o seguinte espaço das funções lineares por partes

Vh:={v:v∈C0⁢(ℐ),v|Ii∈P1⁢(Ii),i=1,2,⁢…,n}. (1.5)

Observamos que toda função v∈Vh é unicamente determinadas por seus valores nodais {αi=v(xi)}i=0n. Reciprocamente, todo conjunto de valores nodas {αi}i=0n determina unicamente uma função v∈Vh. Desta observação, temos que os valores nodais determinam os graus de liberdade com a base nodal {φj}j=0n para Vh definida por

φj⁢(xi)={1,i=j,0,i≠j, (1.6)

com i,j=0,1,⁢…,n. Podemos verificar que

φi⁢(x)={(x-xi-1)/hi,x∈Ii,(xi+1-x)/hi+1,x∈Ii+1,0,noutros casos (1.7)

veja, Figura 1.1. É notável que ϕi⁢(x) tem suporte compacto Ii∪Ii+1.

1.1.1 Interpolação

A interpolação é uma das técnicas de aproximação de funções. Dada uma função contínua f em I=[x0,x1], definimos o operador de interpolação linear π:C0⁢(I)→P1⁢(I) por

π⁢f⁢(x)=f⁢(x0)⁢φ0⁢(x)+f⁢(x1)⁢φ1⁢(x). (1.8)

Observamos que π⁢f é igual a f nos nodos x0 e x1.

Exemplo 1.1.1.

A Figura 1.2 ilustra a interpolação da função f⁢(x)=3⁢sen⁡(2⁢π⁢x) no espaço P1([1/4,3/4)]. Neste caso

π⁢f⁢(x)=f⁢(14)⁢3/4-x1/2+f⁢(34)⁢x-1/41/2. (1.9)

Com o FENiCS, podemos computar a função interpolada π⁢f com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

# malha
mesh = IntervalMesh(4,0.25,0.75)

# espaco
V = FunctionSpace(mesh, ’P’, 1)

# funcao
f = Expression(’3*sin(2*pi*x[0])’,
                   degree=10)

# interpolacao
pif = interpolate(f,V)

# grafico
xx = IntervalMesh(100,0.25,0.75)
plot(f,mesh=xx,label="$f$")
plot(pif,mesh=mesh,
     marker=’o’,label="$\pi f$")
plt.legend(numpoints=1)
plt.grid(’on’)
plt.show()

Agora, vamos buscar medir o erro de interpolação, i.e. f-π⁢f. Para tanto, podemos usar a norma L2 definida por

∥v∥L2⁢(I)=(∫iv2⁢d⁢x)1/2. (1.10)

Lembramos que valem as desigualdades triangular

∥v+w∥L2⁢(I)≤∥v∥L2⁢(I)+∥w∥L2⁢(I) (1.11)

e a de Cauchy-Schwarz11Também conhecida como desigualdade de Cauchy–Bunyakovsky–Schwarz. Augustin-Louis Cauchy, 1789 - 1857, matemático francês. Viktor Yakovlevich Bunyakovsky, 1804 - 1889, matemático Russo. Karl Hermann Amandus Schwarz, 1843 - 1921, matemático alemão.

∫Iv⁢w⁢d⁢x≤∥v∥L2⁢(I)⁢∥w∥L2⁢(I), (1.12)

para qualquer funções v,w∈L2⁢(I).

Proposição 1.1.1.

(Erro da interpolação linear) O interpolador π⁢f satisfaz as estimativas

∥f-π⁢f∥L2⁢(I) ≤C⁢h2⁢∥f′′∥L2⁢(I), (1.13)
∥(f-π⁢f)′∥L2⁢(I) ≤C⁢h⁢∥f′′∥L2⁢(I), (1.14)

onde C é uma constante e h=x1-x0.

Demonstração.

Denotemos o erro de interpolação por e=f-π⁢f. Do teorema fundamental do cálculo, temos

e⁢(y)=e⁢(x0)+∫x0ye′⁢(x)⁢d⁢x, (1.15)

onde e⁢(x0)=f⁢(x0)-π⁢f⁢(x0)=0. Daí, usando a desigualdade de Cauchy-Schwarz (1.12), temos

e⁢(y) =∫x0ye′⁢d⁢x (1.16)
≤∫x0y|e′|⁢d⁢x (1.17)
≤∫I1⋅|e′|⁢d⁢x (1.18)
≤(∫I12⁢d⁢x)1/2⁢(∫Ie′⁣2⁢d⁢x)1/2 (1.19)
=h1/2⁢(∫Ie′⁣2⁢d⁢x)1/2, (1.20)

donde

e⁢(y)2≤h⁢∫Ie′⁣2⁢d⁢x=h⁢∥e′∥L2⁢(I)2. (1.21)

Então, integrando em I obtemos

∥e∥L2⁢(I)2=∫Ie2⁢(y)⁢d⁢y≤∫Ih⁢∥e′∥L2⁢(I)2⁢d⁢y=h2⁢∥e′∥L2⁢(I)2, (1.22)

ou seja,

∥e∥L2⁢(I)≤h⁢∥e′∥L2⁢(I). (1.23)

Agora, observando que e⁢(x0)=e⁢(x1)=0, o teorema de Rolle22Michel Rolle, 1652 - 1719, matemático francês. garante a existência de um ponto x~∈I tal que e′⁢(x~)=0, donde do teorema fundamental do cálculo e da desigualdade de Cauchy-Schwarz, segue

e′⁢(y) =e′⁢(x~)+∫x~ye′′⁢d⁢x (1.24)
=∫x~ye′′⁢d⁢x (1.25)
≤∫I1⋅|e′′|⁢d⁢x (1.26)
≤h1/2⁢(∫Ie′′⁣2)1/2. (1.27)

Então, integrando em I, obtemos

∥e′∥L2⁢(I)2≤h2⁢∥e′′∥L2⁢(I)2, (1.28)

a qual, observando que e′′=f′′, equivale a segunda estimativa procurada, i.e.

∥(f-π⁢f)′∥L2⁢(I)≤C⁢h⁢∥f′′∥L2⁢(I). (1.29)

Por fim, de (1.23) e de (1.23), obtemos a primeira estimativa desejada

∥f-π⁢f∥L2⁢(I)≤C⁢h2⁢∥f′′∥L2⁢(I). (1.30)

∎

Exemplo 1.1.2.

A Figura 1.3 mostra a evolução do erro na norma L2 da interpolação de f⁢(x)=3⁢sen⁡(2⁢π⁢x) no espaço P1([0,h)] para h=10-5⁢,10-4,⋯⁢,10-1.

Com o FENiCS, podemos computar os erros de interpolação com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

# funcao
f = Expression(’3*sin(2*pi*x[0])’,
               degree=10)

n=5
for k in np.arange(1,n+1):
    h = 10**-k
    mesh = IntervalMesh(1,0,h)
    V = FunctionSpace(mesh, ’P’, 1)
    pif = interpolate(f,V)
    e = errornorm(f,pif,’L2’)
    print("%d %1.0E %1.1E" % (k,h,e))

Vamos, agora, generalizar o resultado da Proposição 1.1.1 para a interpolação no espaço Vh das funções lineares por parte. Dada uma função contínua f em I=[l0,l1], definimos o operador interpolador π:C0⁢(I)→Vh na malha ℐ de I por

π⁢f⁢(x)=∑i=0nf⁢(xi)⁢φi⁢(x). (1.31)

O seguinte resultado fornece uma estimativa do erro de interpolação em relação ao tamanho hi de cada elemento da malha.

Proposição 1.1.2.

O interpolador π⁢f satisfaz as estimativas

∥f-π⁢f∥L2⁢(I)2 ≤C⁢∑i=1nhi4⁢∥f′′∥L2⁢(I)2, (1.32)
∥(f-π⁢f)′∥L2⁢(I)2 ≤C⁢∑i=1nhi2⁢∥f′′∥L2⁢(I)2. (1.33)
Demonstração.

Ambas desigualdades seguem da desigualdade triangular e da Proposição 1.1.1. Por exemplo, para a primeira desigualdade, temos

∥f-π⁢f∥L2⁢(I)2 ≤∑i=1n∥f-π⁢f∥L2⁢(Ii)2 (1.35)
≤∑i=1nC⁢hi4⁢∥f′′∥L2⁢(Ii)2. (1.36)

∎

1.1.2 Projeção L2

Dada uma função f∈L2⁢(I), definimos o operador de projeção L2 Ph:L2⁢(I)→Vh por

∫I(f-Ph⁢f)⁢v⁢d⁢x=0, ∀v∈Vh. (1.37)

Como Vh é um espaço de dimensão finita, a condição (1.38) é equivalente a

∫I(f-Ph⁢f)⁢φi⁢d⁢x=0, i=0,1,⁢⋯,n, (1.38)

onde φi é a i-ésima função base de Vh. Além disso, como Ph⁢f∈Vh, temos

Ph⁢f=∑j=0nξj⁢φj, (1.39)

onde ξj, j=0,1,⁢…,n, são n+1 incógnitas a determinar. Logo,

∫I(f-Ph⁢f)⁢φi⁢d⁢x=0 ⇔∫If⁢φi⁢d⁢x=∫IPh⁢f⁢φi⁢d⁢x (1.40)
⇔∫If⁢φi⁢d⁢x=∫I(∑j=0nξj⁢φj)⁢φi⁢d⁢x (1.41)
⇔∑j=0nξj⁢∫Iφj⁢φi⁢d⁢x=∫If⁢φi⁢d⁢x, (1.42)

para i=0,1,⁢…,n.

Observemos, agora, que (1.42) consiste em um sistema de n+1 equações lineares para as n+1 incógnitas ξj, j=0,1,⁢…,n. Este, por sua vez, pode ser escrito na seguinte forma matricial

M⁢ξ=b, (1.43)

onde M=[mi,j]i,j=0n+1 é chamada de matriz de massa

mi,j=∫Iφj⁢φi⁢d⁢x (1.44)

e b=(b0,b1,…,bn) é chamado de vetor de carregamento

bi=∫If⁢φi⁢d⁢x. (1.45)

Ou seja, a projeção L2 de f no espaço Vh é

Ph⁢f=∑j=0nξj⁢φj, (1.46)

onde ξ=(ξ0,ξ1,…,ξn) é solução do sistema (1.43).

Exemplo 1.1.4.

A Figura 1.5 ilustra a projeção L2 da função f⁢(x)=3⁢sen⁡(2⁢π⁢x) no espaço Vh das funções lineares por partes em uma malha uniforme do intervalo I=[1/4,3/4] com n=4 subintervalos (5 pontos).

Com o FENiCS, podemos computar Ph⁢f com o seguinte código:

from __future__ import print_function, division
from fenics import *
import numpy as np
import matplotlib.pyplot as plt

# malha
mesh = IntervalMesh(4,0.25,0.75)

# espaco
V = FunctionSpace(mesh, ’P’, 1)

# funcao
f = Expression(’3*sin(2*pi*x[0])’,
                   degree=10)

# projecao
Phf = project(f, V)

# grafico
xx = IntervalMesh(100,0.25,0.75)
plot(f,mesh=xx,label="$f$")
plot(Phf,mesh=mesh,
     marker=’o’,label="$P_h f$")
plt.legend(numpoints=1)
plt.grid(’on’)
plt.show()

O próximo teorema mostra que Ph⁢f é a função que melhor aproxima f dentre todas as funções do espaço Vh.

Teorema 1.1.1.

(A melhor aproximação.) A projeção L2 satisfaz

∥f-Ph⁢f∥L2⁢(I)≤∥f-v∥L2⁢(I),∀v∈Vh. (1.47)
Demonstração.

Dado v∈Vh, temos

∥f-Ph⁢f∥L2⁢(I)2 =∫I|f-Ph⁢f|2⁢d⁢x (1.48)
=∫I(f-Ph⁢f)⁢(f-v+v-Ph⁢f)⁢d⁢x (1.49)
=∫I(f-Ph⁢f)⁢(f-v)⁢d⁢x+∫I(f-Ph⁢f)⁢(v-Ph⁢f)⁢d⁢x (1.50)
=∫I(f-Ph⁢f)⁢(f-v)⁢d⁢x (1.51)
≤∥f-Ph⁢f∥L2⁢(I)⁢∥f-v∥L2⁢(I), (1.52)

donde segue o resultado. ∎

O próximo teorema fornece uma estimativa a-priori do erro ∥f-Ph⁢f∥L2⁢(I) em relação ao tamanho da malha.

Teorema 1.1.2.

A projeção L2 satisfaz

∥f-Ph⁢f∥L2⁢(I)≤C⁢∑i=1nhi4⁢∥f′′∥L2⁢(Ii)2. (1.53)
Demonstração.

Tomando a interpolação π⁢f∈Vh, temos do Teorema da melhor aproximação (Teorema 1.1.1) e da estimativa do erro de interpolação (Proposição 1.1.2) que

∥f-Ph⁢f∥L2⁢(I)2 ≤∥f-π⁢f∥L2⁢(I)2 (1.54)
≤C⁢∑i=1nhi4⁢∥f′′∥L2⁢(Ii)2. (1.55)

∎
