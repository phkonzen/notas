Cálculo com Python¶

Nesta terceira parte do curso vamos estudar como usar o Python para computar derivadas de funes de uma varivel.

3 - Derivadas¶

Aqui, vamor aprender a usar a biblioteca Python de matemtica simblica Sympy para computar:

  • derivadas

  • retas tangentes

  • derivadas de ordem mais altas

  • mximos e mnimos

Comeamos carrendo a biblioteca SymPy:

In [1]:
from sympy import *
init_printing()
var('x,y')
Out[1]:
$$\left ( x, \quad y\right )$$

Para fixar as ideias, vamos continuar trabalhando com a funo:

$f(x) = (x^3 - 3x + 2)e^{-x/4} - 1$.

In [2]:
f = Lambda(x, (x**3 - 3*x + 2)*exp(-x/4) - 1)
f
Out[2]:
$$\left( x \mapsto \left(x^{3} - 3 x + 2\right) e^{- \frac{x}{4}} - 1 \right)$$

Derivadas¶

Vamos ver como podemos usar o SymPy para computarmos a derivada da funo $f$, i.e.

$\displaystyle f' = \frac{d f}{d x}$.

Para tanto, usamos a funo diff:

In [3]:
diff(f(x), x)
Out[3]:
$$\left(3 x^{2} - 3\right) e^{- \frac{x}{4}} - \frac{e^{- \frac{x}{4}}}{4} \left(x^{3} - 3 x + 2\right)$$

Para avaliar a derivada em um ponto, por exemplo, para calcular $f'(1)$, digitamos:

In [4]:
diff(f(x),x).subs(x,1)
Out[4]:
$$0$$

Tambm, podemos definir a funo derivada de $f$ com o seguinte cdigo:

In [5]:
fl = Lambda(x, diff(f(x), x))
fl
Out[5]:
$$\left( x \mapsto \left(3 x^{2} - 3\right) e^{- \frac{x}{4}} - \frac{e^{- \frac{x}{4}}}{4} \left(x^{3} - 3 x + 2\right) \right)$$

Com isso, $f'(1)$ pode ser computada com:

In [6]:
fl(1)
Out[6]:
$$0$$

Reta Tangente¶

Aqui, vamos ver como computar a reta tangente ao grfico da funo $f$ no ponto $\displaystyle x_0 = -\frac{1}{2}$. Lembremos que tal reta tangente tem equao:

$y = f'(x_0)(x - x_0) + f(x_0)$.

Assim sendo, podemos definir a funo affim cujo grfico a reta tangente com o seguinte comando:

In [8]:
x0 = -1/2
r = Lambda(x, fl(x0)*(x-x0) + f(x0))
r
Out[8]:
$$\left( x \mapsto - 3.50567802667549 x + 1.07153701576279 \right)$$

Vejamos os grficos de $f(x)$ e da reta tangente computada.

In [9]:
%matplotlib inline
p = plot(f(x), (x,-2,2), line_color='blue', show=False)
q = plot(r(x), (x,-1.5,1), line_color='red', show=False)
p.extend(q)
p.show()

$\blacktriangleleft$

Derivadas de ordem mais altas¶

Similarmente a derivadas de primeira ordem, as derivadas de ordem mais alta podem ser obtidas com a funo diff. Por exemplo, para computarmos

$\displaystyle f'' = \frac{d^2 f}{d x^2}$

digitamos:

In [11]:
diff(f(x), x, 2)
Out[11]:
$$\frac{e^{- \frac{x}{4}}}{16} \left(x^{3} - 24 x^{2} + 93 x + 26\right)$$

Máximos e Mínimos Locais¶

Agora, vamos ver como podemos usar o que aprendemos at aqui para resolver problemas de mximos e mnimos. Nosso objetivo ser encontrar e determinar os pontos de mximos e mnimos locais da funo $f$. Para tanto, vamos usar o teste da segunda derivada.

Comecemos lembrando do grfico de $f$:

In [13]:
plot(f(x), (x,-2,2))
plot(f(x), (x,2,40))
plot(f(x), (x,40,80))
Out[13]:
<sympy.plotting.plot.Plot at 0x7f45765fe6a0>

Observando os grficos, podemos inferir que $f$ tem mximos locais prximos a $x=-1$ e $x=10$ e um mnimo local prximo do ponto $x=1$. Logo, podemos usar estes valores como parmetro da funo nsolve para resolvermos

$f'(x) = 0$.

Computamos:

In [14]:
#ponto crtico prx. de x=-1
x1 = nsolve(fl(x), x, -1)
x1
Out[14]:
$$-1.15206734782504$$
In [15]:
#ponto crtico prx. de x=1
x2 = nsolve(fl(x), x, 1)
x2
Out[15]:
$$1.0$$
In [16]:
#ponto crtico prx. de x=10
x3 = nsolve(fl(x), x, 10)
x3
Out[16]:
$$12.152067347825$$

Ok. Acamos de encontrar os pontos crticos de $f(x)$ (podem haver outros pontos crticos?). Por inspeso grfica, vemos que $x_1 \approx -1,15$ e $x_3 \approx 12,15$ so os pontos de mximos locais de $f(x)$, com $x_3$ ponto de mximo global? O ponto $x_2 = 1$ ponto de mnimo local?

Podemos responder parcialmente estas questes com o teste da segunda derivada. Vejamos:

In [17]:
# f''
fll = Lambda(x, diff(f(x), x, 2))
fll
Out[17]:
$$\left( x \mapsto \frac{e^{- \frac{x}{4}}}{16} \left(x^{3} - 24 x^{2} + 93 x + 26\right) \right)$$
In [18]:
fll(x1)
Out[18]:
$$-9.54699341615203$$
In [19]:
fll(x2)
Out[19]:
$$4.67280469842843$$
In [20]:
fll(x3)
Out[20]:
$$-1.77782118521895$$

Logo, pelo teste da segunda derivada, confirmamos que $x_1 \approx -1,15$ e $x_3 \approx 12,15$ so os pontos de mximos locais e $x_2=1$ ponto de mnimo local. As demais questes em aberto ficam de exerccio!
