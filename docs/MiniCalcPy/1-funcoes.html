Cálculo com Python¶

Este um minicurso sobre resoluo de problemas de clculo diferencial e integral usando Python e a biblioteca de matemtica simblica Sympy. No necessrio ter qualquer experincia na linguagem de programao Python para seguir este curso.

Neste minicurso voc vai aprender a usar o Python para:

  • manipular e graficar funes de uma varivel;
  • calcular limites;
  • calcular derivadas;
  • calcular integrais definidas e indefinidas.

Se voc um iniciante em Python no necessrio instalar nenhum programa em seu computador para seguir este curso. O material do curso est disponvel em formatos IPYNB (Jupyter NoteBook) e em HTML. Tambm, est disponvel a seguinte ligao Binder:

Clicando na ligao acima, voc tem acesso a um Jupyter NoteBook online com o material do minicurso.

1 - Funções de uma variável¶

Nesta primeira parte do minicurso Clculo com Python voc aprender a:

Definindo funções¶

Vamos ver como definir uma funo $y = f(x)$ para ser utilizada mais tarde. Ao longo minicurso, vamos considerar a funo:

$f(x) = (x^3 - 3x + 2)e^{-x/4} - 1$.

No Python os operadores aritmticos bsicos so:

+ adio
- subtrao
* multiplico
/ diviso
** potenciao

Tambm, muitas funes elementares j esto definidas por padro no Sympy. Veja a lista aqui!

Para definirmos $f$, podemos usar a funo Lambda como segue:

In [4]:
f = Lambda(x, (x**3 - 3*x + 2)*exp(-x/4) - 1)
f
Out[4]:
$$\left( x \mapsto \left(x^{3} - 3 x + 2\right) e^{- \frac{x}{4}} - 1 \right)$$

Com isso, para computar $f(1)$ digitamos:

In [5]:
f(1)
Out[5]:
$$-1$$

Note que a sada aparece na linha abaixo da entrada $f(1)$. Isto , $f(1) = -1$. Podemos, inclusive utilizar argumentos simblicos, como:

In [6]:
f(x-2)
Out[6]:
$$\left(- 3 x + \left(x - 2\right)^{3} + 8\right) e^{- \frac{x}{4} + \frac{1}{2}} - 1$$

Aqui, a sada significa que $f(x-2) = (-3x + (x-2)^3 + 8)e^{-\frac{x}{4} + \frac{1}{2}} - 1$, o que pode ser imediatamente verificado.

Exemplo:¶

Dada $g(x) = 2 - \sqrt{x^2 - 1}$, encontre $g(1)$, $g(0)$ e $g(2t)$.

Soluo:

In [7]:
g = Lambda(x, 2 - sqrt(x**2 - 1))
g
Out[7]:
$$\left( x \mapsto - \sqrt{x^{2} - 1} + 2 \right)$$
In [8]:
g(1)
Out[8]:
$$2$$
In [9]:
g(0)
Out[9]:
$$2 - i$$

Note que o sympy assume o domnio e a imagem da funo como sendo um subconjunto dos nmeros complexos. Alm disso, antes de calcularmos $g(2t)$ precisamos definir $t$ como uma varivel simblica. Para que $t$ seja tratado como uma varivel real, usamos o seguinte comando:

In [10]:
var('t', real=True)
Out[10]:
$$t$$

Finalmente, computamos $g(2t)$ com:

In [11]:
g(2*t)
Out[11]:
$$- \sqrt{4 t^{2} - 1} + 2$$

$\blacktriangleleft$

Esboço do Gráfico¶

Para plotar o grfico de uma funo $f$ podemos usar o comando plot. Por exemplo, para a funo $f$ definida anteriormente, temos:

In [13]:
%matplotlib inline
plot(f(x))
Out[13]:
<sympy.plotting.plot.Plot at 0x7fd603c792b0>

  • Observao 1: A "linha mgica" %matplotlib inline faz com que o grfico aparea na mesma janela do ambiente. Sem esta linha, uma janela grfica externa ser aberta, quando disponvel.

  • Observao 2: O comando plot(f(x)) tenta advinhar uma caixa grfica adequada. Podemos alterar a caixa, informando o intervalo dos pontos de abscissa que gostaramos de ver. Por exemplo:

In [14]:
plot(f(x), (x, -2, 2))
Out[14]:
<sympy.plotting.plot.Plot at 0x7fd603c52780>

Zeros de uma Função¶

O SymPy conta com a funo solve para computar a soluo de $f(x) = 0$. Por exemplo:

In [16]:
solve(x**2-1, x)
Out[16]:
$$\left [ -1, \quad 1\right ]$$

resolve $x^2 -1 = 0$. Entretanto a funo solve no resolve diretamente a equao $f(x) = 0$ definida anteriormente. Veja o que ocorre:

In [17]:
solve(f(x), x)

No algorithms are implemented to solve equation (x**3 - 3*x + 2)*exp(-x/4) - 1

A ltima linha da mensagem de erro nos diz que o solve no contm um algoritmo para resolver $f(x) = 0$, no caso da $f(x)$ dada. Alternativamente, podemos usar a funo nsolve que usa um algoritmo numrico para encontrar uma soluo para o problema.

Vejamos como usar esta funo nsolve:

In [18]:
nsolve(f(x), x, -2)
Out[18]:
$$-1.92796526009401$$
In [19]:
nsolve(f(x), x, 0)
Out[19]:
$$0.31645451718964$$
In [20]:
nsolve(f(x), x, 2)
Out[20]:
$$1.64337127911688$$

Na verdade, esta funo $f(x)$ tem, ainda, um zero $x \approx 45,\!9162$. Discutiremos isso mais detalhadamente na prxima parte deste minicurso, quando estudarmos a computao de limites com o SymPy.
