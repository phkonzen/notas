2.1 Método de Newton

Consideremos o seguinte problema: F:D⊂ℝn→ℝn encontrar x*∈ℝn tal que

F⁢(x*)=0. (2.1)

Salvo explicitado ao contrário, assumiremos que F∈C1⁢(D), i.e. F é uma função continuamente diferenciável. Vamos, também, denotar por JF⁢(x)=[ȷi,j⁢(x)]i,j=1n,n a matriz jacobiana3434endnote: 34Carl Gustav Jakob Jacobi, 1804 - 1851, matemático alemão. Fonte: Wikipédia. com

ȷi,j⁢(x)=∂fi⁢(x)∂xj, (2.2)

onde F⁢(x)=(f1⁢(x),f2⁢(x),…,fn⁢(x)) e x=(x1,x2,…,xn).

A iteração básica do Método de Newton3636endnote: 363535endnote: 35Isaac Newton, 1642 - 1727, matemático, físico, astrônomo, teólogo e autor inglês. Fonte: Wikipédia: Isaac Newton. consiste em: dada uma aproximação inicial x(0)∈ℝn,

resolverJF⁢(x(k))⁢δ(k)=−F⁢(x(k)) (2.3)
calcularx(k+1)=x(k)+δ(k) (2.4)

para k=0,1,2,… até convergência x(k)→x*.

Observação 2.1.1.

Para x(0) suficientemente próximo da solução x*, o Método de Newton é quadraticamente convergente. Mais precisamente, este resultado de convergência local requer que JF−1⁢(x*) seja não singular e que JF:ℝn→ℝn×n seja Lipschitz3737endnote: 37Rudolf Otto Sigismund Lipschitz, 1832 - 1903, matemático alemão. Fonte: Wikipédia. contínua. Consulte [Quarteroni2007, Seção 7.1] para mais detalhes.

Exemplo 2.1.1.

Consideremos a Equação de Burgers3838endnote: 38Johannes Martinus Burgers, 1895 - 1981, físico holandês. Fonte: Wikipedia.

∂u∂t+u⁢∂u∂x=ν⁢∂2u∂x2 (2.5)

com ν>0, condição inicial

u⁢(0,x)=sin⁡(π⁢x) (2.6)

e condições de contorno de Dirichlet3939endnote: 39Johann Peter Gustav Lejeune Dirichlet, 1805 - 1859, matemático alemão. Fonte: Wikipédia. homogêneas

u⁢(t⁢,0)=u⁢(t⁢,1)=0. (2.7)

Aplicando o Método de Rothe4040endnote: 40Erich Hans Rothe, 1895 - 1988, matemático alemão. Fonte: Wikipédia. com aproximação de Euler4141endnote: 41Leonhard Paul Euler, 1707 - 1783, matemático e físico suíço. Fonte: Wikipédia. implícita, obtemos

u⁢(t+ht,x)−u⁢(t,x)ht+u⁢(t+ht,x)⁢ux⁢(t+ht,x)≈ν⁢ux⁢x⁢(t+ht,x), (2.8)

onde ht>0 é o passo no tempo. Agora, aplicamos diferenças finitas para obter

u⁢(t+ht,xi)−u⁢(t,xi)ht+u⁢(t+ht,xi)⁢u⁢(t+ht,xi+1)−u⁢(t+ht,xi)hx (2.9)
≈ν⁢u⁢(t+ht,xi−1)−2⁢u⁢(t+ht,xi)+u⁢(t+ht,xi+1)hx2, (2.10)

onde, xi=(i−1)⁢hx, i=1,…,nx e hx=1/(nx−1) é o tamanho da malha.

Rearranjando os termos e denotando wi(k)≈u⁢(tk,xi), tk=(k−1)⁢h, obtemos o seguinte problema discreto

w1(k+1)=0 (2.11)
1ht⁢wi(k+1)−1ht⁢wi(k)+1hx⁢wi(k+1)⁢wi+1(k+1)−1hx⁢wi(k+1)⁢wi(k+1) (2.12)
−νhx2⁢wi−1(k+1)+2⁢νhx2⁢wi(k+1)−νhx⁢wi+1(k+1)=0, (2.13)
wnx(k+1)=0, (2.14)

sendo wi(0)=sen⁡(π⁢xi), i=1,…,nx e k=1,2,….

Este problema pode ser reescrito como segue: para cada k=1,2,…, encontrar v∈ℝnx, tal que

F⁢(v;v(0))=0, (2.15)

onde vi≈wi(k+1), vi(0)≈wi(k) e

f1⁢(v,v(0))=v1, (2.16)
fi⁢(v;v(0))=1ht⁢vi−1ht⁢vi(0)+1hx⁢vi⁢vi+1−1hx⁢vi⁢vi (2.17)
−νhx2⁢vi−1+2⁢νhx2⁢vi−νhx⁢vi+1, (2.18)
fnx⁢(v;v(0))=vnx. (2.19)

A matriz jacobiana associada J=[ȷi,j]i,jnx,nx contém

ȷi,j=0,j≠i−1,i,i+1, (2.20)
ȷ1,1=1, (2.21)
ȷ1,2=0, (2.22)
ȷi,i−1=−νhx2, (2.23)
ȷi,i=1ht+1hx⁢vi+1−2hx⁢vi+2⁢νhx2, (2.24)
ȷi,i+1=1hx⁢vi−νhx2, (2.25)
ȷnx,nx−1=0 (2.26)
ȷnx,nx=1. (2.27)
E. 2.1.1.

Considere o problema discreto apresentado no Exemplo 2.1.1 para diferentes valores do coeficiente de difusão ν=1.,0.5,0.1,0.01,0.001. Simule o problema com cada uma das seguintes estratégias e as compare quanto ao desempenho computacional.

  1. a)

    Simule-o aplicando o Método de Newton com o solver linear npla.solve.

  2. b)

    Observe que a jacobiana é uma matriz tridiagonal. Simule o problema aplicando o Método de Newton com o solver linear npla.solve_banded.

  3. c)

    Aloque a jacobiana como uma matriz esparsa. Então, simule o problema aplicando o Método de Newton com solver linear adequado para matrizes esparsas.
